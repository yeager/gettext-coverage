# Gettext Coverage

Show translation coverage per package in a distribution.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install gettext-coverage
```
